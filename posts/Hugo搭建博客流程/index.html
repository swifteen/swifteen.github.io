    搭建HUGO博客

    April 17, 2022
    工具
    HUGO, blog

    搭建HUGO博客 #

    安装hugo #

    进入 release页面下载，选择下载 hugo_extended_0.97.0_Linux-64bit.deb带extended后缀的安装包

    sudo dpkg -i hugo_extended_0.97.0_Linux-64bit.deb

    创建hugo工程 # 

    mkdir ~/Public/Book
cd ~/Public/Book
hugo new site ./

    下载主题 # 

    git init
git submodule add https://github.com/alex-shpak/hugo-book themes/hugo-book
cp -R themes/hugo-book/exampleSite/content .

    运行Web服务 # 

    ubuntu:~/Public/Book$ hugo server --minify --theme hugo-book --bind="0.0.0.0" -p 8888
